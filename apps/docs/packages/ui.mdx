# @novai/ui

The UI package provides a comprehensive set of beautiful, accessible components built with shadcn/ui and Tailwind CSS. These components are designed specifically for NovAI applications and provide a consistent, modern design system.

## Installation

```bash
npm install @novai/ui
# or
pnpm add @novai/ui
```

## Quick Start

```typescript
import { Button, Card, Input, Dialog } from '@novai/ui';

// Basic component usage
function GameInterface() {
  return (
    <Card>
      <Card.Header>
        <Card.Title>Adventure Log</Card.Title>
      </Card.Header>
      <Card.Content>
        <p>You stand at the entrance to the ancient dungeon...</p>
      </Card.Content>
      <Card.Footer>
        <Button>Continue Adventure</Button>
      </Card.Footer>
    </Card>
  );
}
```

## Performance Optimization

### Lazy Loading

```typescript
import { lazy, Suspense } from 'react';
import { Skeleton } from '@novai/ui';

const CharacterSheet = lazy(() => import('@novai/ui').then(m => ({ default: m.CharacterSheet })));

<Suspense fallback={<Skeleton className="h-96" />}>
  <CharacterSheet />
</Suspense>
```

### Memoization

```typescript
import { memo } from 'react';
import { Card, CardContent } from '@novai/ui';

const CharacterCard = memo(({ character }) => (
  <Card>
    <CardContent>
      <h3>{character.name}</h3>
      <p>Level {character.level} {character.class}</p>
    </CardContent>
  </Card>
));
```

## Testing

### Component Testing

```typescript
import { render, screen } from '@testing-library/react';
import { Button } from '@novai/ui';

describe('Button Component', () => {
  it('renders with correct text', () => {
    render(<Button>Click me</Button>);
    expect(screen.getByRole('button', { name: 'Click me' })).toBeInTheDocument();
  });

  it('handles click events', () => {
    const handleClick = jest.fn();
    render(<Button onClick={handleClick}>Click me</Button>);
    
    screen.getByRole('button').click();
    expect(handleClick).toHaveBeenCalled();
  });
});
```

### Integration Testing

```typescript
import { render, screen, fireEvent } from '@testing-library/react';
import { Dialog, DialogTrigger, DialogContent } from '@novai/ui';

describe('Dialog Integration', () => {
  it('opens and closes correctly', () => {
    render(
      <Dialog>
        <DialogTrigger>Open</DialogTrigger>
        <DialogContent>Content</DialogContent>
      </Dialog>
    );

    fireEvent.click(screen.getByText('Open'));
    expect(screen.getByText('Content')).toBeInTheDocument();

    fireEvent.click(screen.getByRole('button', { name: /close/i }));
    expect(screen.queryByText('Content')).not.toBeInTheDocument();
  });
});
```

### Button

#### Props

```typescript
interface ButtonProps {
  variant?: 'default' | 'destructive' | 'outline' | 'secondary' | 'ghost' | 'link';
  size?: 'default' | 'sm' | 'lg' | 'icon';
  disabled?: boolean;
  children: React.ReactNode;
  onClick?: () => void;
}
```

  children: React.ReactNode;
